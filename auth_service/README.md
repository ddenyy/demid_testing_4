# auth_service

Сервис аунтификации пользователя

- auth_service_pb2_grpc и auth_service_pb2 - сгенерированные proto-файлы
- code_database_user - реализует логику работы с БД пользователей
- user_service.py — реализация gRPC-сервисов
- password — файлы по хешированию и энтропии паролей.
- main - точка запуска сервиса
- requirements - все необходимые зависимости
