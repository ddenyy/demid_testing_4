# Assembly_PC

Система для сборки и конфигурации персональных компьютеров.

##  Архитектура системы
- assembly_db - база сборок
- assembly_service - сервис сборки
- auth_service - сервис аунтификации
- user_db - база данных пользователя
- componets_db - база данных компонентов
- components_service - сервер для сборки компонетов из Json
- app_ui - UI интерфейс к приложению
- grpcwebproxy* - мост между grpc-web и grpc - сервисами
- proto - определение grpc сервисов


## Описание

Система автоматической сборки конфигураций персональных компьютеров. Пользователь регистрируется в системе, входит в свой аккаунт и по нажатию одной кнопки получает готовую сборку ПК

## Инструкция по запуску
- клонирование репозитория
- поднятие контейнеров (docker-cmpose up/ docker-cmpose up -d)
- [сервис доступен по порту 8081](http://localhost:8081/)
