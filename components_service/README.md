# assembly_service

Сервис сборок

- components_pb2_grpc и acomponents_pb2 - сгенерированные proto-файлы
- code_database - реализует логику работы с БД компонентов
- componets_service.py — реализация gRPC сервиса
- main - точка запуска сервиса
- components.json - стартовые данные
- requirements - все необходимые зависимости
